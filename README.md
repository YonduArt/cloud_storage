# Cloud Storage

Fullstack-приложение для хранения и управления файлами: личное файловое пространство, папки, корзина, публичные ссылки, предпросмотр файлов, история действий, квоты и поиск.

## Стек

- Backend: Java 17, Spring Boot, Spring Security, JWT, Spring Data JPA, Flyway, PostgreSQL
- Frontend: React, TypeScript, React Router, Axios, Vite
- Infra: Docker Compose, Nginx

## Возможности

- регистрация и авторизация пользователей
- загрузка, скачивание, переименование, перемещение и удаление файлов
- работа с папками и загрузка вложенных директорий
- корзина с восстановлением и окончательным удалением
- публичные ссылки на файлы и папки
- предпросмотр изображений, PDF, видео, аудио и текстовых файлов
- история действий и статистика занятого места
- поиск по файлам, папкам и содержимому
